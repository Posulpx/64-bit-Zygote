# Phenomena — Emergent Behaviors

## Definition

A **phenomenon** is an observed system-level behavior that is not explicitly programmed into any single cell or gene. Phenomena emerge from the interaction of cells over time.

## How to Document a Phenomenon

1. Describe the experimental setup (genome, environment).
2. Show the relevant signal trace.
3. Name the phenomenon.
4. Hypothesize the mechanism.
5. Design a test to confirm or refute.

## Desired Phenomena (Wishlist)

- **Signal routing** — cells self-organize into feed-forward pipelines.
- **Homeostasis** — system maintains a steady signal level despite input variability.
- **Pattern memory** — signal pattern persists after input is removed.
- **Division of labor** — cell types specialize through competition.
- **Novelty detection** — system responds differently to familiar vs. novel inputs.
